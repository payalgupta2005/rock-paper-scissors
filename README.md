# 🎮 Rock Paper Scissors Game  

A simple Rock Paper Scissors game built with **HTML, CSS, and JavaScript**.  

## 🚀 How to Play
- Choose Rock, Paper, or Scissors.  
- The computer makes a random choice.  
- See who wins! 🎉  

## 📂 Project Structure
- `index.html` → Game layout  
- `style.css` → Styling  
- `script.js` → Game logic
